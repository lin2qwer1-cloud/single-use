Why · What

무엇을 푸는 엔진인가

최종 리시버가 받은 금액의 "출처 전수 분해"

코스트센터 배부(assessment · distribution · indirect activity allocation)가 여러 사이클을 거치고 나면, 최종 리시버에 쌓인 금액이 애초에 어느 오브젝트의 어느 원가요소에서 출발했는지는 표준 리포트로 알 수 없다. 이 엔진은 한 회계월의 배부 전기를 실행 순서대로 재생하여, 모든 최종 잔액을 (원천 오브젝트, 원천 원가요소) → (최종 오브젝트, 최종 원가요소) 단위로 분해한 전수 테이블을 만든다.

핵심 설계 결정 두 가지. 첫째, 사이클 마스터(T811C/S)는 사후 변경될 수 있으므로 참조하지 않는다 — 실제 전기 이력인 T811D 실행 인덱스와 CO 명세서만 믿는다. 둘째, 프루닝으로 버린 잔량조차 PRUNED 행으로 방출하여 in = out 대사를 항상 성립시킨다.

The Core

전파 알고리즘

propagate_origins — 이 엔진의 심장

세 가지 이동 규칙

A
계정보존 — Distribution
RKIV/RKPV, RKIB. 원가요소가 유지된 채 이동하므로 풀을 (sender, 원가요소) 단위로 잡고, 같은 원가요소의 state만 비례 이동.
B
가중평균 — Assessment
RKIU/RKPU와 아웃플로우. 어세스먼트 원가요소로 뭉쳐 나가므로 풀은 sender 전체 — 모든 state가 금액 비중대로 함께 이동.
U
추적불가 — Untraceable
PAROB가 비었거나 무의미(길이 ≤ 2)해 sender를 모르는 엣지. 리시버 앵커의 UNTRACEABLE state로 정직하게 등록.

이동 수식

exec_seq가 같은 엣지 묶음마다 (A 먼저, B 다음):

pool(sender[, kstar]) = Σ state(sender[, kstar])           " 규칙별 풀
이동분  = state.amount × edge.amount / pool                  " 원천별 비례 배분
잔여분  = state.amount × (pool − sent) / pool                " sender에 남는 몫

이동한 조각은 원천 속성(origin_objnr / origin_kstar / origin_type / origin_vrgng)을 그대로 들고 리시버의 state가 된다. 이후 라운드에서 리시버가 다시 sender가 되면 그 조각이 또 이동한다 — 다단 배부가 자연스럽게 원천까지 연결되는 이유다.

pool = 0 폴백 — 자기참조 금지 (2026-07-23 수정)

엣지가 있는데 sender의 state가 비어 있으면(이미 전부 배부됐거나 음수 상쇄) 비례 배분이 불가능하다. 이때 이동액은 UNTRACEABLE로 방출하되 origin은 반드시 sender로 남긴다. 초기 구현은 origin을 receiver로 찍었고, 그 결과 IDES 2018년 데이터에서 #COPA → #COPA = 123,601.17 같은 자기참조 터미널 행이 나왔다 — "터미널이 자기 자신을 원천으로 갖는 행은 무조건 버그"라는 검증 규칙으로 잡아낸 결함. 수정 후 같은 데이터셋에서 자기참조 0건, 모든 UNTRACEABLE 행이 실제 sender를 보여준다.

프루닝과 대사 불변식

라운드마다 state를 키(오브젝트·현재원가요소·원천 4속성)로 압축하고, |잔액| < ε(0.01)인 조각은 계산에서 제외한다. 단, 버리지 않고 origin_type='PRUNED' 행으로 출력에 방출한다. 그래서:

ev_in  = Σ 자기비용 + Σ UNTRACEABLE 유입      " 엔진에 들어온 총량
ev_out = Σ et_trace.amount                     " 엔진에서 나간 총량
불변식: ev_in = ev_out  " PRUNED 방출 덕에 항상 성립 — 리포트 헤더에 diff 표기

내부 계산은 P LENGTH 16 DECIMALS 6(amt6)으로 하고 출력만 2자리로 반올림한다. 반올림 잔차는 final_objnr 버킷별로 그 버킷의 최대 금액 행에 몰아 보정한다 — 전역 1회 보정이던 초기 구현은 #COPA 버킷이 원본 전표 대비 0.02 어긋났고, 버킷별 보정으로 바꾼 뒤 센트 단위까지 일치한다(158,525.57).

구현 디테일 하나: EMPTY KEY 표준 테이블에는 COLLECT가 실패하므로 압축·집계는 전부 HASHED 테이블 수동 집계로 구현되어 있다(실측 확인된 제약).

입력 가드

get_origin_trace는 WRTTP로 '04'(실적) 외의 모든 값을 명시적으로 '01'(계획)로 강제한다. 이전에는 '99' 같은 값이 조용히 계획 분기로 흘러 trace 행 라벨까지 오염시켰다. 실적 경로는 VERSN을 '000'으로 고정한다.

METHOD make_exec_seq.
  DATA lv_dc TYPE c LENGTH 8.
  DATA lv_tc TYPE c LENGTH 6.
  lv_dc = iv_cpudt.
  lv_tc = iv_cputm.
  rv_seq = CONV int8( lv_dc ) * 1000000 + CONV int8( lv_tc ).

METHOD derive_rule.
  DATA(lv_parob) = condense( CONV string( iv_parob ) ).

METHOD get_actual_edges.

METHOD get_plan_edges.
  SELECT d~cycle, k~cpudt, k~cputm,
         c~vrgng, c~parob, c~objnr, c~kstar,
         c~wkg001, c~wkg002, c~wkg003, c~wkg004, c~wkg005, c~wkg006,
         c~wkg007, c~wkg008, c~wkg009, c~wkg010, c~wkg011, c~wkg012
    FROM t811d AS d
    INNER JOIN coej AS c
      ON  c~kokrs = d~nkrsub AND c~gjahr = d~gjahr AND c~belnr = d~docnr
    INNER JOIN cobk AS k
      ON  k~kokrs = c~kokrs AND k~belnr = c~belnr AND k~gjahr = c~gjahr
    WHERE d~tab = 'CCSS' AND d~nkrsub = @iv_kokrs
      AND d~gjahr = @iv_gjahr AND d~perio = @iv_poper AND d~reversed = @space
      AND c~wrttp = '01' AND c~versn = @iv_versn AND c~beknz = 'S'
      AND c~vrgng IN ( 'RKPU', 'RKPV' )
    INTO TABLE @DATA(lt_raw).

  DATA(lv_comp) = |WKG{ iv_poper }|.
  DATA lt_agg TYPE HASHED TABLE OF ty_edge
       WITH UNIQUE KEY cycle exec_seq vrgng rule sender receiver kstar.
  LOOP AT lt_raw ASSIGNING FIELD-SYMBOL(<r>).
    ASSIGN COMPONENT lv_comp OF STRUCTURE <r> TO FIELD-SYMBOL(<amt>).
    IF <amt> IS NOT ASSIGNED OR <amt> = 0. CONTINUE. ENDIF.
    DATA(ls_key) = VALUE ty_edge(
      perio = iv_poper cycle = <r>-cycle
      exec_seq = make_exec_seq( iv_cpudt = <r>-cpudt iv_cputm = <r>-cputm )
      vrgng = <r>-vrgng
      rule  = derive_rule( iv_parob = <r>-parob iv_vrgng = <r>-vrgng+3(1) )
      sender = <r>-parob receiver = <r>-objnr kstar = <r>-kstar ).
    READ TABLE lt_agg ASSIGNING FIELD-SYMBOL(<e>)
         WITH TABLE KEY cycle = ls_key-cycle exec_seq = ls_key-exec_seq
                        vrgng = ls_key-vrgng rule = ls_key-rule
                        sender = ls_key-sender receiver = ls_key-receiver
                        kstar = ls_key-kstar.
    IF sy-subrc <> 0.
      INSERT ls_key INTO TABLE lt_agg ASSIGNING <e>.
    ENDIF.
    <e>-amount = <e>-amount + <amt>.
    UNASSIGN <amt>.
  ENDLOOP.
  LOOP AT lt_agg INTO DATA(ls_e) WHERE amount <> 0.
    APPEND ls_e TO rt_edge.
  ENDLOOP.
ENDMETHOD.
METHOD get_self_costs.
  CLEAR: et_self, et_outflow.
  TYPES: BEGIN OF ty_line,
           objnr  TYPE c LENGTH 22,
           kstar  TYPE c LENGTH 10,
           vrgng  TYPE c LENGTH 4,
           cpudt  TYPE d,
           cputm  TYPE t,
           amount TYPE ty_amt,
         END OF ty_line.
  DATA lt_lines TYPE STANDARD TABLE OF ty_line WITH EMPTY KEY.

  IF iv_wrttp = '04'.
    SELECT c~objnr, c~kstar, c~vrgng, k~cpudt, k~cputm, c~wkgbtr AS amount
      FROM coep AS c
      INNER JOIN cobk AS k
        ON k~kokrs = c~kokrs AND k~belnr = c~belnr AND k~gjahr = c~gjahr
      WHERE c~kokrs = @iv_kokrs AND c~gjahr = @iv_gjahr AND c~perio = @iv_poper
        AND c~wrttp = '04' AND c~versn = '000'
        AND c~vrgng NOT IN ('RKIU','RKIV','RKIB')
      INTO CORRESPONDING FIELDS OF TABLE @lt_lines.
  ELSE.
    SELECT c~objnr, c~kstar, c~vrgng, k~cpudt, k~cputm,
           c~wkg001, c~wkg002, c~wkg003, c~wkg004, c~wkg005, c~wkg006,
           c~wkg007, c~wkg008, c~wkg009, c~wkg010, c~wkg011, c~wkg012
      FROM coej AS c
      INNER JOIN cobk AS k
        ON k~kokrs = c~kokrs AND k~belnr = c~belnr AND k~gjahr = c~gjahr
      WHERE c~kokrs = @iv_kokrs AND c~gjahr = @iv_gjahr
        AND c~wrttp = '01' AND c~versn = @iv_versn
        AND c~vrgng NOT IN ('RKPU','RKPV')
      INTO TABLE @DATA(lt_cj).
    DATA(lv_comp) = |WKG{ iv_poper }|.
    LOOP AT lt_cj ASSIGNING FIELD-SYMBOL(<cj>).
      ASSIGN COMPONENT lv_comp OF STRUCTURE <cj> TO FIELD-SYMBOL(<amt>).
      IF <amt> IS NOT ASSIGNED OR <amt> = 0. CONTINUE. ENDIF.
      APPEND VALUE ty_line( objnr = <cj>-objnr kstar = <cj>-kstar vrgng = <cj>-vrgng
                            cpudt = <cj>-cpudt cputm = <cj>-cputm amount = <amt> ) TO lt_lines.
      UNASSIGN <amt>.
    ENDLOOP.
  ENDIF.

  DATA lt_of TYPE HASHED TABLE OF ty_edge
       WITH UNIQUE KEY exec_seq vrgng sender kstar receiver.
  DATA lt_sf TYPE HASHED TABLE OF ty_self
       WITH UNIQUE KEY objnr kstar vrgng origin_type.
  LOOP AT lt_lines INTO DATA(ls_l).
    DATA(lv_is_out) = xsdbool( ( ls_l-vrgng = 'KSPA' OR ls_l-vrgng = 'KSOS'
                              OR ls_l-vrgng = 'KOAO' OR ls_l-vrgng = 'KOAL' )
                              AND ls_l-amount < 0 ).
    IF lv_is_out = abap_true.
      DATA(ls_ok) = VALUE ty_edge(
        perio = iv_poper cycle = space
        exec_seq = make_exec_seq( iv_cpudt = ls_l-cpudt iv_cputm = ls_l-cputm )
        vrgng = ls_l-vrgng rule = 'B'
        sender = ls_l-objnr
        receiver = COND #( WHEN ls_l-vrgng = 'KSPA' OR ls_l-vrgng = 'KSOS'
                           THEN '#COPA' ELSE '#SETTLEMENT' )
        kstar = ls_l-kstar ).
      READ TABLE lt_of ASSIGNING FIELD-SYMBOL(<of>)
           WITH TABLE KEY exec_seq = ls_ok-exec_seq vrgng = ls_ok-vrgng
                          sender = ls_ok-sender kstar = ls_ok-kstar
                          receiver = ls_ok-receiver.
      IF sy-subrc <> 0.
        INSERT ls_ok INTO TABLE lt_of ASSIGNING <of>.
      ENDIF.
      <of>-amount = <of>-amount - ls_l-amount.
    ELSE.
      DATA(ls_s) = VALUE ty_self(
        perio = iv_poper objnr = ls_l-objnr kstar = ls_l-kstar vrgng = ls_l-vrgng
        origin_type = COND #( WHEN ls_l-vrgng = 'RKL'  OR ls_l-vrgng = 'RKLN'
                                OR ls_l-vrgng = 'KZPI' OR ls_l-vrgng = 'KOAO'
                                OR ls_l-vrgng = 'KOAL' OR ls_l-vrgng = 'KSPA'
                                OR ls_l-vrgng = 'KSOS' OR ls_l-vrgng = 'KAMV'
                              THEN 'EXTERNAL' ELSE 'PRIMARY' ) ).
      READ TABLE lt_sf ASSIGNING FIELD-SYMBOL(<sf>)
           WITH TABLE KEY objnr = ls_s-objnr kstar = ls_s-kstar
                          vrgng = ls_s-vrgng origin_type = ls_s-origin_type.
      IF sy-subrc <> 0.
        INSERT ls_s INTO TABLE lt_sf ASSIGNING <sf>.
      ENDIF.
      <sf>-amount = <sf>-amount + ls_l-amount.
    ENDIF.
  ENDLOOP.
  LOOP AT lt_sf INTO DATA(ls_s2) WHERE amount <> 0.
    APPEND ls_s2 TO et_self.
  ENDLOOP.
  LOOP AT lt_of INTO DATA(ls_o).
    APPEND ls_o TO et_outflow.
  ENDLOOP.
ENDMETHOD.
METHOD propagate_origins.
  CLEAR: et_trace, ev_in, ev_out.
  DATA: lt_state TYPE tt_state,
        lt_out   TYPE tt_state,
        lv_in6   TYPE ty_amt6.

  LOOP AT it_self INTO DATA(ls_s).
    APPEND VALUE ty_state( obj = ls_s-objnr cur_kstar = ls_s-kstar
                           origin_objnr = ls_s-objnr origin_kstar = ls_s-kstar
                           origin_type = ls_s-origin_type origin_vrgng = ls_s-vrgng
                           amount = ls_s-amount ) TO lt_state.
  ENDLOOP.
  LOOP AT it_edge INTO DATA(ls_e) WHERE rule = 'U'.
    APPEND VALUE ty_state( obj = ls_e-receiver cur_kstar = ls_e-kstar
                           origin_objnr = ls_e-receiver origin_kstar = ls_e-kstar
                           origin_type = 'UNTRACEABLE' origin_vrgng = ls_e-vrgng
                           amount = ls_e-amount ) TO lt_state.
  ENDLOOP.
  LOOP AT lt_state INTO DATA(ls_st).
    lv_in6 = lv_in6 + ls_st-amount.
  ENDLOOP.

  DATA lt_seq TYPE SORTED TABLE OF int8 WITH UNIQUE KEY table_line.
  LOOP AT it_edge INTO ls_e WHERE rule = 'A' OR rule = 'B'.
    INSERT ls_e-exec_seq INTO TABLE lt_seq.
  ENDLOOP.

  TYPES: BEGIN OF ty_pool,
           obj   TYPE c LENGTH 22,
           kstar TYPE c LENGTH 10,
           sent  TYPE ty_amt6,
           pool  TYPE ty_amt6,
         END OF ty_pool.
  DATA lt_pool TYPE HASHED TABLE OF ty_pool WITH UNIQUE KEY obj kstar.
  DATA(lv_neg) = CONV ty_amt6( 0 - iv_eps ).

  LOOP AT lt_seq INTO DATA(lv_seq).
    DO 2 TIMES.
      DATA(lv_rule) = COND char1( WHEN sy-index = 1 THEN 'A' ELSE 'B' ).
      CLEAR lt_pool.
      LOOP AT it_edge INTO ls_e WHERE exec_seq = lv_seq AND rule = lv_rule.
        DATA(lv_pk) = COND #( WHEN lv_rule = 'A' THEN ls_e-kstar ELSE space ).
        READ TABLE lt_pool ASSIGNING FIELD-SYMBOL(<p>)
             WITH TABLE KEY obj = ls_e-sender kstar = lv_pk.
        IF sy-subrc <> 0.
          INSERT VALUE ty_pool( obj = ls_e-sender kstar = lv_pk ) INTO TABLE lt_pool ASSIGNING <p>.
        ENDIF.
        <p>-sent = <p>-sent + ls_e-amount.
      ENDLOOP.
      IF lt_pool IS INITIAL. CONTINUE. ENDIF.

      LOOP AT lt_state INTO ls_st.
        READ TABLE lt_pool ASSIGNING <p> WITH TABLE KEY obj = ls_st-obj
             kstar = COND #( WHEN lv_rule = 'A' THEN ls_st-cur_kstar ELSE space ).
        IF sy-subrc = 0. <p>-pool = <p>-pool + ls_st-amount. ENDIF.
      ENDLOOP.

      DATA lt_new TYPE tt_state.
      CLEAR lt_new.
      LOOP AT it_edge INTO ls_e WHERE exec_seq = lv_seq AND rule = lv_rule.
        READ TABLE lt_pool INTO DATA(ls_p) WITH TABLE KEY obj = ls_e-sender
             kstar = COND #( WHEN lv_rule = 'A' THEN ls_e-kstar ELSE space ).
        IF ls_p-pool = 0.
          " pool empty: keep the SENDER as origin so terminals (#COPA etc.)
          " never own themselves as origin - reads as "untraceable balance
          " of <sender> flowed to <receiver>"
          APPEND VALUE ty_state( obj = ls_e-receiver cur_kstar = ls_e-kstar
                                 origin_objnr = ls_e-sender origin_kstar = ls_e-kstar
                                 origin_type = 'UNTRACEABLE' origin_vrgng = ls_e-vrgng
                                 amount = ls_e-amount ) TO lt_new.
          lv_in6 = lv_in6 + ls_e-amount.
        ELSE.
          LOOP AT lt_state INTO ls_st WHERE obj = ls_e-sender.
            IF lv_rule = 'A' AND ls_st-cur_kstar <> ls_e-kstar. CONTINUE. ENDIF.
            APPEND VALUE ty_state( obj = ls_e-receiver cur_kstar = ls_e-kstar
                                   origin_objnr = ls_st-origin_objnr
                                   origin_kstar = ls_st-origin_kstar
                                   origin_type  = ls_st-origin_type
                                   origin_vrgng = ls_st-origin_vrgng
                                   amount = ls_st-amount * ls_e-amount / ls_p-pool ) TO lt_new.
          ENDLOOP.
        ENDIF.
      ENDLOOP.

      LOOP AT lt_state ASSIGNING FIELD-SYMBOL(<st>).
        READ TABLE lt_pool INTO ls_p WITH TABLE KEY obj = <st>-obj
             kstar = COND #( WHEN lv_rule = 'A' THEN <st>-cur_kstar ELSE space ).
        IF sy-subrc = 0 AND ls_p-pool <> 0.
          <st>-amount = <st>-amount * ( ls_p-pool - ls_p-sent ) / ls_p-pool.
        ENDIF.
      ENDLOOP.
      APPEND LINES OF lt_new TO lt_state.
    ENDDO.

    " compress via manual HASHED aggregation (COLLECT fails on EMPTY KEY tt_state - confirmed)
    DATA lt_cmp TYPE HASHED TABLE OF ty_state
         WITH UNIQUE KEY obj cur_kstar origin_objnr origin_kstar origin_type origin_vrgng.
    CLEAR lt_cmp.
    LOOP AT lt_state INTO ls_st.
      READ TABLE lt_cmp ASSIGNING FIELD-SYMBOL(<c>)
           WITH TABLE KEY obj = ls_st-obj cur_kstar = ls_st-cur_kstar
                          origin_objnr = ls_st-origin_objnr origin_kstar = ls_st-origin_kstar
                          origin_type = ls_st-origin_type origin_vrgng = ls_st-origin_vrgng.
      IF sy-subrc <> 0.
        INSERT ls_st INTO TABLE lt_cmp ASSIGNING <c>.
      ELSE.
        <c>-amount = <c>-amount + ls_st-amount.
      ENDIF.
    ENDLOOP.
    DELETE lt_cmp WHERE amount = 0.
    CLEAR lt_state.
    LOOP AT lt_cmp INTO ls_st.
      IF ls_st-amount < iv_eps AND ls_st-amount > lv_neg.
        ls_st-origin_objnr = ls_st-obj.
        ls_st-origin_kstar = ls_st-cur_kstar.
        ls_st-origin_type  = 'PRUNED'.
        APPEND ls_st TO lt_out.
      ELSE.
        APPEND ls_st TO lt_state.
      ENDIF.
    ENDLOOP.
  ENDLOOP.

  " final aggregation (manual HASHED, same reason), 2dp rounding,
  " residual correction PER FINAL OBJECT so terminal buckets (#COPA,
  " #SETTLEMENT) match their source documents to the cent
  DATA lt_hi TYPE HASHED TABLE OF ty_state
       WITH UNIQUE KEY obj cur_kstar origin_objnr origin_kstar origin_type origin_vrgng.
  LOOP AT lt_state INTO ls_st.
    READ TABLE lt_hi ASSIGNING FIELD-SYMBOL(<h>)
         WITH TABLE KEY obj = ls_st-obj cur_kstar = ls_st-cur_kstar
                        origin_objnr = ls_st-origin_objnr origin_kstar = ls_st-origin_kstar
                        origin_type = ls_st-origin_type origin_vrgng = ls_st-origin_vrgng.
      IF sy-subrc <> 0.
        INSERT ls_st INTO TABLE lt_hi ASSIGNING <h>.
      ELSE.
        <h>-amount = <h>-amount + ls_st-amount.
      ENDIF.
  ENDLOOP.
  LOOP AT lt_out INTO ls_st.
    READ TABLE lt_hi ASSIGNING <h>
         WITH TABLE KEY obj = ls_st-obj cur_kstar = ls_st-cur_kstar
                        origin_objnr = ls_st-origin_objnr origin_kstar = ls_st-origin_kstar
                        origin_type = ls_st-origin_type origin_vrgng = ls_st-origin_vrgng.
      IF sy-subrc <> 0.
        INSERT ls_st INTO TABLE lt_hi ASSIGNING <h>.
      ELSE.
        <h>-amount = <h>-amount + ls_st-amount.
      ENDIF.
  ENDLOOP.
  DELETE lt_hi WHERE amount = 0.

  TYPES: BEGIN OF ty_res,
           obj     TYPE c LENGTH 22,
           hi_sum  TYPE ty_amt6,
           rnd_sum TYPE ty_amt6,
           max_abs TYPE ty_amt6,
           max_idx TYPE i,
         END OF ty_res.
  DATA lt_res TYPE HASHED TABLE OF ty_res WITH UNIQUE KEY obj.
  LOOP AT lt_hi INTO ls_st.
    DATA(lv_r2) = CONV ty_amt( ls_st-amount ).
    APPEND VALUE ty_trace(
        perio = iv_poper
        origin_objnr = ls_st-origin_objnr origin_kstar = ls_st-origin_kstar
        final_objnr = ls_st-obj final_kstar = ls_st-cur_kstar
        amount = lv_r2
        origin_type = ls_st-origin_type origin_vrgng = ls_st-origin_vrgng
        final_type = SWITCH #( ls_st-obj WHEN '#COPA'       THEN 'TO_COPA'
                                         WHEN '#SETTLEMENT' THEN 'TO_SETTLEMENT'
                                         ELSE 'OBJECT' ) ) TO et_trace.
    READ TABLE lt_res ASSIGNING FIELD-SYMBOL(<rs>) WITH TABLE KEY obj = ls_st-obj.
    IF sy-subrc <> 0.
      INSERT VALUE ty_res( obj = ls_st-obj ) INTO TABLE lt_res ASSIGNING <rs>.
    ENDIF.
    <rs>-hi_sum  = <rs>-hi_sum  + ls_st-amount.
    <rs>-rnd_sum = <rs>-rnd_sum + lv_r2.
    IF abs( ls_st-amount ) > <rs>-max_abs.
      <rs>-max_abs = abs( ls_st-amount ).
      <rs>-max_idx = lines( et_trace ).
    ENDIF.
  ENDLOOP.
  LOOP AT lt_res INTO DATA(ls_res).
    DATA(lv_resid) = CONV ty_amt( ls_res-hi_sum - ls_res-rnd_sum ).
    IF lv_resid <> 0 AND ls_res-max_idx > 0.
      READ TABLE et_trace ASSIGNING FIELD-SYMBOL(<t>) INDEX ls_res-max_idx.
      <t>-amount = <t>-amount + lv_resid.
    ENDIF.
  ENDLOOP.
  DELETE et_trace WHERE amount = 0.
  ev_in  = lv_in6.
  LOOP AT et_trace INTO DATA(ls_t).
    ev_out = ev_out + ls_t-amount.
  ENDLOOP.
ENDMETHOD.
METHOD get_origin_trace.
    CLEAR: et_trace, ev_in, ev_out.
    DATA lt_edge TYPE tt_edge.
    " input guard: only '04' (actual) and '01' (plan) are supported -
    " anything else is coerced to '01' explicitly so trace rows never
    " carry an unsupported value type label
    DATA(lv_wrttp) = COND ty_c2( WHEN iv_wrttp = '04' THEN '04' ELSE '01' ).
    DATA(lv_versn) = COND versn( WHEN lv_wrttp = '04' THEN '000' ELSE iv_versn ).
    IF lv_wrttp = '04'.
      lt_edge = get_actual_edges( iv_kokrs = iv_kokrs iv_gjahr = iv_gjahr iv_poper = iv_poper ).
    ELSE.
      lt_edge = get_plan_edges( iv_kokrs = iv_kokrs iv_gjahr = iv_gjahr
                                iv_poper = iv_poper iv_versn = lv_versn ).
    ENDIF.
    get_self_costs( EXPORTING iv_kokrs = iv_kokrs iv_gjahr = iv_gjahr iv_poper = iv_poper
                              iv_versn = lv_versn iv_wrttp = lv_wrttp
                    IMPORTING et_self = DATA(lt_self) et_outflow = DATA(lt_of) ).
    APPEND LINES OF lt_of TO lt_edge.
    propagate_origins( EXPORTING it_edge = lt_edge it_self = lt_self
                                 iv_poper = iv_poper iv_eps = '0.01'
                       IMPORTING et_trace = DATA(lt_tr) ev_in = ev_in ev_out = ev_out ).
    LOOP AT lt_tr INTO DATA(ls_t).
      ls_t-kokrs = iv_kokrs. ls_t-gjahr = iv_gjahr.
      ls_t-versn = lv_versn. ls_t-wrttp = lv_wrttp.
      APPEND ls_t TO et_trace.
    ENDLOOP.
  ENDMETHOD.
ENDCLASS.

CLASS ycl_co_cca_trace_rpt IMPLEMENTATION.
  METHOD derive_attrs.
    DATA lt_objnr TYPE SORTED TABLE OF j_objnr WITH UNIQUE KEY table_line.
    LOOP AT it_trace INTO DATA(ls_t).
      IF ls_t-origin_objnr(1) <> '#'.
        INSERT CONV j_objnr( ls_t-origin_objnr ) INTO TABLE lt_objnr.
      ENDIF.
      IF ls_t-final_objnr(1) <> '#'.
        INSERT CONV j_objnr( ls_t-final_objnr ) INTO TABLE lt_objnr.
      ENDIF.
    ENDLOOP.
    LOOP AT lt_objnr INTO DATA(lv_objnr).
      DATA(ls_a) = VALUE ty_attr( objnr = lv_objnr otype = lv_objnr(2) okey = lv_objnr+2 ).
      CASE lv_objnr(2).
        WHEN 'KS'.
          DATA(lv_kostl) = CONV kostl( lv_objnr+6(10) ).
          ls_a-okey = lv_kostl.
          SELECT SINGLE bukrs, gsber FROM csks
            WHERE kokrs = @iv_kokrs AND kostl = @lv_kostl
              AND datab <= @iv_keydate AND datbi >= @iv_keydate
            INTO ( @ls_a-bukrs, @ls_a-gsber ).
          SELECT SINGLE ktext FROM cskt
            WHERE spras = @sy-langu AND kokrs = @iv_kokrs
              AND kostl = @lv_kostl AND datbi >= @iv_keydate
            INTO @ls_a-text.
        WHEN 'OR'.
          SELECT SINGLE bukrs, gsber, ktext FROM aufk
            WHERE objnr = @lv_objnr INTO ( @ls_a-bukrs, @ls_a-gsber, @ls_a-text ).
        WHEN 'PR'.
          SELECT SINGLE pbukr, pgsbr, post1 FROM prps
            WHERE objnr = @lv_objnr INTO ( @ls_a-bukrs, @ls_a-gsber, @ls_a-text ).
        WHEN OTHERS.
      ENDCASE.
      INSERT ls_a INTO TABLE rt_attr.
    ENDLOOP.
  ENDMETHOD.

  METHOD filter_trace.
    CLEAR: et_trace, ev_sum.
    LOOP AT it_trace INTO DATA(ls_t).
      DATA ls_fin TYPE ty_attr. DATA ls_org TYPE ty_attr.
      CLEAR: ls_fin, ls_org.
      READ TABLE it_attr INTO ls_fin WITH TABLE KEY objnr = CONV j_objnr( ls_t-final_objnr ).
      READ TABLE it_attr INTO ls_org WITH TABLE KEY objnr = CONV j_objnr( ls_t-origin_objnr ).
      DATA(lv_ok) = SWITCH abap_bool( iv_mode
        WHEN c_mode_final  THEN xsdbool( ls_fin-bukrs IN it_bukrs AND ls_fin-gsber IN it_gsber )
        WHEN c_mode_origin THEN xsdbool( ls_org-bukrs IN it_bukrs AND ls_org-gsber IN it_gsber )
        WHEN c_mode_both   THEN xsdbool( ls_fin-bukrs IN it_bukrs AND ls_fin-gsber IN it_gsber
                                     AND ls_org-bukrs IN it_bukrs AND ls_org-gsber IN it_gsber )
        ELSE abap_true ).
      IF lv_ok = abap_true.
        APPEND ls_t TO et_trace.
        ev_sum = ev_sum + ls_t-amount.
      ENDIF.
    ENDLOOP.
  ENDMETHOD.
ENDCLASS.

METHOD get_actual_edges BY DATABASE PROCEDURE FOR HDB LANGUAGE SQLSCRIPT OPTIONS READ-ONLY
  USING v_coep_view t811d cobk.

  -- posted cycle run index (reversed runs excluded)
  lt_docs = SELECT DISTINCT nkrsub AS kokrs, gjahr, perio, docnr, cycle
            FROM t811d
            WHERE mandt = :iv_mandt AND tab = 'CCSS'
              AND nkrsub = :iv_kokrs AND gjahr = :iv_gjahr
              AND perio BETWEEN :iv_perio_from AND :iv_perio_to
              AND reversed = '';

  -- edges from receiver lines (BEKNZ='S'): sender=PAROB, receiver=OBJNR
  et_edge = SELECT c.perio,
                   d.cycle,
                   CAST(k.cpudt AS BIGINT) * 1000000 + CAST(k.cputm AS BIGINT) AS exec_seq,
                   c.vrgng,
                   CASE WHEN LENGTH(RTRIM(c.parob)) <= 2 THEN 'U'
                        WHEN c.vrgng = 'RKIU'            THEN 'B'
                        ELSE 'A' END AS rule,
                   c.parob AS sender,
                   c.objnr AS receiver,
                   c.kstar,
                   SUM(c.wkgbtr) AS amount
            FROM v_coep_view AS c
            INNER JOIN :lt_docs AS d
              ON  d.kokrs = c.kokrs AND d.gjahr = c.gjahr
              AND d.perio = c.perio AND d.docnr = c.belnr
            INNER JOIN cobk AS k
              ON  k.mandt = :iv_mandt AND k.kokrs = c.kokrs
              AND k.belnr = c.belnr   AND k.gjahr = c.gjahr
            WHERE c.mandt = :iv_mandt
              AND c.wrttp = '04' AND c.versn = '000'
              AND c.vrgng IN ('RKIU','RKIV','RKIB')
              AND c.beknz = 'S'
              AND k.stflg = '' AND k.stokz = ''
            GROUP BY c.perio, d.cycle, k.cpudt, k.cputm, c.vrgng, c.parob, c.objnr, c.kstar
            HAVING SUM(c.wkgbtr) <> 0;

METHOD propagate_origins BY DATABASE PROCEDURE FOR HDB LANGUAGE SQLSCRIPT OPTIONS READ-ONLY.
  DECLARE lv_i INT;
  DECLARE lv_n INT;
  DECLARE lv_seq BIGINT;
  DECLARE lv_add DECIMAL(34,6);
  DECLARE lv_evin DECIMAL(34,6);

  lt_state = SELECT perio, objnr AS obj, kstar AS cur_kstar,
                    objnr AS origin_objnr, kstar AS origin_kstar,
                    origin_type, vrgng AS origin_vrgng,
                    CAST(amount AS DECIMAL(34,6)) AS amount
             FROM :it_self;

  lt_state = SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount
             FROM :lt_state
             UNION ALL
             SELECT perio, receiver AS obj, kstar AS cur_kstar,
                    receiver AS origin_objnr, kstar AS origin_kstar,
                    'UNTRACEABLE' AS origin_type, vrgng AS origin_vrgng,
                    CAST(amount AS DECIMAL(34,6)) AS amount
             FROM :it_edge WHERE rule = 'U';

  SELECT COALESCE(SUM(amount), 0) INTO lv_evin FROM :lt_state;

  lt_out = SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount
           FROM :lt_state WHERE 1 = 0;

  lt_seq = SELECT DISTINCT exec_seq FROM :it_edge WHERE rule = 'A' OR rule = 'B' ORDER BY exec_seq ASC;
  SELECT COUNT(*) INTO lv_n FROM :lt_seq;

  FOR lv_i IN 0..lv_n-1 DO
    SELECT exec_seq INTO lv_seq FROM :lt_seq ORDER BY exec_seq ASC LIMIT 1 OFFSET :lv_i;

    lt_ea = SELECT * FROM :it_edge WHERE exec_seq = :lv_seq AND rule = 'A';
    lt_sent_a = SELECT perio, sender, kstar, SUM(amount) AS sent
                FROM :lt_ea GROUP BY perio, sender, kstar;
    lt_pool_a = SELECT s.perio, s.obj, s.cur_kstar, SUM(s.amount) AS pool
                FROM :lt_state AS s
                INNER JOIN :lt_sent_a AS t
                  ON t.perio = s.perio AND t.sender = s.obj AND t.kstar = s.cur_kstar
                GROUP BY s.perio, s.obj, s.cur_kstar;

    lt_mov_a = SELECT e.perio, e.receiver AS obj, e.kstar AS cur_kstar,
                      s.origin_objnr, s.origin_kstar, s.origin_type, s.origin_vrgng,
                      CAST(ROUND(CAST(s.amount AS DOUBLE) * CAST(e.amount AS DOUBLE)
                                 / CAST(p.pool AS DOUBLE), 6) AS DECIMAL(34,6)) AS amount
               FROM :lt_ea AS e
               INNER JOIN :lt_pool_a AS p
                 ON p.perio = e.perio AND p.obj = e.sender AND p.cur_kstar = e.kstar
               INNER JOIN :lt_state AS s
                 ON s.perio = e.perio AND s.obj = e.sender AND s.cur_kstar = e.kstar
               WHERE p.pool <> 0;

    lt_mov_a0 = SELECT e.perio, e.receiver AS obj, e.kstar AS cur_kstar,
                       e.receiver AS origin_objnr, e.kstar AS origin_kstar,
                       'UNTRACEABLE' AS origin_type, e.vrgng AS origin_vrgng,
                       CAST(e.amount AS DECIMAL(34,6)) AS amount
                FROM :lt_ea AS e
                LEFT JOIN :lt_pool_a AS p
                  ON p.perio = e.perio AND p.obj = e.sender AND p.cur_kstar = e.kstar
                WHERE p.pool IS NULL OR p.pool = 0;

    lt_state = SELECT s.perio, s.obj, s.cur_kstar, s.origin_objnr, s.origin_kstar,
                      s.origin_type, s.origin_vrgng,
                      CAST(CASE WHEN p.pool IS NULL OR p.pool = 0 THEN CAST(s.amount AS DOUBLE)
                                ELSE ROUND(CAST(s.amount AS DOUBLE)
                                     * (CAST(p.pool AS DOUBLE) - CAST(t.sent AS DOUBLE))
                                     / CAST(p.pool AS DOUBLE), 6) END AS DECIMAL(34,6)) AS amount
               FROM :lt_state AS s
               LEFT JOIN :lt_sent_a AS t
                 ON t.perio = s.perio AND t.sender = s.obj AND t.kstar = s.cur_kstar
               LEFT JOIN :lt_pool_a AS p
                 ON p.perio = s.perio AND p.obj = s.obj AND p.cur_kstar = s.cur_kstar
               UNION ALL
               SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount FROM :lt_mov_a
               UNION ALL
               SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount FROM :lt_mov_a0;

    SELECT COALESCE(SUM(amount), 0) INTO lv_add FROM :lt_mov_a0;
    lv_evin = :lv_evin + :lv_add;

    lt_eb = SELECT * FROM :it_edge WHERE exec_seq = :lv_seq AND rule = 'B';
    lt_sent_b = SELECT perio, sender, SUM(amount) AS sent
                FROM :lt_eb GROUP BY perio, sender;
    lt_pool_b = SELECT s.perio, s.obj, SUM(s.amount) AS pool
                FROM :lt_state AS s
                INNER JOIN :lt_sent_b AS t
                  ON t.perio = s.perio AND t.sender = s.obj
                GROUP BY s.perio, s.obj;

    lt_mov_b = SELECT e.perio, e.receiver AS obj, e.kstar AS cur_kstar,
                      s.origin_objnr, s.origin_kstar, s.origin_type, s.origin_vrgng,
                      CAST(ROUND(CAST(s.amount AS DOUBLE) * CAST(e.amount AS DOUBLE)
                                 / CAST(p.pool AS DOUBLE), 6) AS DECIMAL(34,6)) AS amount
               FROM :lt_eb AS e
               INNER JOIN :lt_pool_b AS p
                 ON p.perio = e.perio AND p.obj = e.sender
               INNER JOIN :lt_state AS s
                 ON s.perio = e.perio AND s.obj = e.sender
               WHERE p.pool <> 0;

    lt_mov_b0 = SELECT e.perio, e.receiver AS obj, e.kstar AS cur_kstar,
                       e.receiver AS origin_objnr, e.kstar AS origin_kstar,
                       'UNTRACEABLE' AS origin_type, e.vrgng AS origin_vrgng,
                       CAST(e.amount AS DECIMAL(34,6)) AS amount
                FROM :lt_eb AS e
                LEFT JOIN :lt_pool_b AS p
                  ON p.perio = e.perio AND p.obj = e.sender
                WHERE p.pool IS NULL OR p.pool = 0;

    lt_state = SELECT s.perio, s.obj, s.cur_kstar, s.origin_objnr, s.origin_kstar,
                      s.origin_type, s.origin_vrgng,
                      CAST(CASE WHEN p.pool IS NULL OR p.pool = 0 THEN CAST(s.amount AS DOUBLE)
                                ELSE ROUND(CAST(s.amount AS DOUBLE)
                                     * (CAST(p.pool AS DOUBLE) - CAST(t.sent AS DOUBLE))
                                     / CAST(p.pool AS DOUBLE), 6) END AS DECIMAL(34,6)) AS amount
               FROM :lt_state AS s
               LEFT JOIN :lt_sent_b AS t
                 ON t.perio = s.perio AND t.sender = s.obj
               LEFT JOIN :lt_pool_b AS p
                 ON p.perio = s.perio AND p.obj = s.obj
               UNION ALL
               SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount FROM :lt_mov_b
               UNION ALL
               SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount FROM :lt_mov_b0;

    SELECT COALESCE(SUM(amount), 0) INTO lv_add FROM :lt_mov_b0;
    lv_evin = :lv_evin + :lv_add;

    lt_state = SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar,
                      origin_type, origin_vrgng,
                      CAST(SUM(amount) AS DECIMAL(34,6)) AS amount
               FROM :lt_state
               GROUP BY perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng
               HAVING SUM(amount) <> 0;

    lt_out = SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount FROM :lt_out
             UNION ALL
             SELECT perio, obj, cur_kstar, obj AS origin_objnr, cur_kstar AS origin_kstar,
                    'PRUNED' AS origin_type, origin_vrgng, amount
             FROM :lt_state WHERE ABS(amount) < :iv_eps;

    lt_state = SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount
               FROM :lt_state WHERE ABS(amount) >= :iv_eps;
  END FOR;

  lt_all = SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount FROM :lt_out
           UNION ALL
           SELECT perio, obj, cur_kstar, origin_objnr, origin_kstar, origin_type, origin_vrgng, amount FROM :lt_state;

  -- high-precision aggregation, then 2-dec rounding with per-perio residual correction
  lt_hi = SELECT perio, origin_objnr, origin_kstar, obj, cur_kstar, origin_type, origin_vrgng,
                 CAST(SUM(amount) AS DECIMAL(34,6)) AS amount
          FROM :lt_all
          GROUP BY perio, origin_objnr, origin_kstar, obj, cur_kstar, origin_type, origin_vrgng
          HAVING SUM(amount) <> 0;

  lt_rnd = SELECT perio, origin_objnr, origin_kstar, obj, cur_kstar, origin_type, origin_vrgng,
                  CAST(ROUND(amount, 2) AS DECIMAL(34,6)) AS amount,
                  ROW_NUMBER() OVER (PARTITION BY perio ORDER BY ABS(amount) DESC) AS rn
           FROM :lt_hi;

  lt_resid = SELECT perio, CAST(SUM(amount) - SUM(ROUND(amount, 2)) AS DECIMAL(34,6)) AS resid
             FROM :lt_hi
             GROUP BY perio;

  et_trace = SELECT '' AS kokrs, '0000' AS gjahr, r.perio, '' AS versn, '' AS wrttp,
                    r.origin_objnr, r.origin_kstar,
                    r.obj AS final_objnr, r.cur_kstar AS final_kstar,
                    CASE WHEN r.rn = 1 THEN ROUND(r.amount + COALESCE(d.resid, 0), 2) ELSE r.amount END AS amount,
                    r.origin_type, r.origin_vrgng,
                    CASE r.obj WHEN '#COPA'       THEN 'TO_COPA'
                               WHEN '#SETTLEMENT' THEN 'TO_SETTLEMENT'
                               ELSE 'OBJECT' END AS final_type
             FROM :lt_rnd AS r
             LEFT JOIN :lt_resid AS d
               ON d.perio = r.perio;

  ev_in = ROUND(:lv_evin, 2);
  SELECT COALESCE(SUM(amount), 0) INTO ev_out FROM :et_trace;
ENDMETHOD.
